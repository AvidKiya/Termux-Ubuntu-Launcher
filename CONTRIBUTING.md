# Contributing

Contributions are welcome.

## فارسی

برای مشارکت:

1. پروژه را Fork کنید.
2. یک Branch جدید بسازید.
3. تغییرات را Commit کنید.
4. Pull Request بفرستید.

لطفاً اسکریپت‌ها را تا حد امکان POSIX/Bash-friendly و مناسب Termux نگه دارید.

## English

To contribute:

1. Fork the project.
2. Create a new branch.
3. Commit your changes.
4. Open a pull request.

Please keep scripts Termux-friendly and easy to read.
